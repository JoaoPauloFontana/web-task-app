## Instalação 
### Requisitos
 - Docker
 - Porta 8012 liberada.

Clone the project:
```
git clone https://github.com/JoaoPauloFontana/web-task-app.git
```

Execute um dos seguintes comandos na pasta raiz do projeto:

bash
```
bash install.sh
```
ou
CMD
```
install
```
Endereço de acesso: http://localhost:3000

## Tecnologias utilizadas
 - Yarn
 - Docker
